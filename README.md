# 🚀 Millennium Falcon 3D Viewer

Un proyecto web interactivo desarrollado con **Three.js** que permite visualizar el icónico modelo 3D del **Millennium Falcon** en un entorno espacial con estrellas animadas y rotación automática del modelo.

Este proyecto demuestra el uso de **renderizado 3D en la web**, carga de modelos **GLTF**, iluminación, controles de cámara y efectos visuales para crear una experiencia interactiva tipo demo espacial.

---

## 🌌 Vista del Proyecto

Visualización del modelo 3D con:

* ⭐ Fondo de estrellas animadas
* 🛸 Rotación automática del modelo
* 🎮 Controles de cámara interactivos
* 💡 Iluminación dinámica con sombras
* ⚡ Renderizado optimizado con Three.js

---

## 🛠️ Tecnologías Utilizadas
